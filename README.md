NetwUx is a homemade project mad just for fun, the main goal of it is to share a small text-only page with other users inside the terminal of every linux machine!

It's easy to use, inside the .zip you'll find the instructions :)

To let you understand better, here is an example.

When you use the command netwux.web , the terminal will be like a search bar of google where you put the link (netwux.name.domain) and you can navigate inside that page and read the content. Soon I'll add images and much more, for now is only text.

The links can be found by searching netwux.archive.files
